# BC Housing Affordability Dashboard (2000–2025)

## Overview
A 25-year analysis of housing affordability across British Columbia 
using official BC Data Catalogue (BC Stats) data — covering median 
home prices, family income, mortgage rates, and affordability index 
across ~30 BC regions.

## Key Findings
- Provincial average: families spend ~30% of income on mortgage payments
- Vancouver is consistently the least affordable market in BC
- Affordability worsened sharply post-2020, partially recovered 2022-23, 
  then declined again through 2025

## Tools Used
- Microsoft Power BI Desktop
- Power Query (data transformation, custom date modeling)
- DAX (Avg Median Price, Avg Affordability Index, Total Resales)
- Data source: BC Data Catalogue — Housing Affordability for Homeowners in B.C.
